## About The Project


This app uses react and fetch to GET and display API data for the first 151 pokemon. From there the data is transformed and displayed all using react.js.

<p align="right">(<a href="#top">back to top</a>)</p>



### Built With

Built with React, styling using Material UI, and API data from PokeAPI

* [React.js](https://reactjs.org/)
* [Material-UI](https://mui.com/)
* [PokeAPI](https://pokeapi.co/)
